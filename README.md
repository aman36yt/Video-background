# 🎥 Video Background Web Page

  A modern and responsive web page with a fullscreen video background, preloader, and interactive controls 🚀
---

## ✨ Features

🎬 Fullscreen video background
⏳ Preloader animation for smooth loading
⏯️ Play & Pause button for video control
📱 Fully responsive design using Bootstrap
🎨 Clean and modern UI

---

## 🛠️ Tech Stack

* 🧱 HTML5 – Structure
* 🎨 CSS3 – Styling
* ⚡ JavaScript – Functionality
* 📦 Bootstrap – Responsive layout

---

## ⚙️ How It Works

* The page loads with a **preloader animation**
* Once loaded, a **video plays in the background**
* Users can **pause or play the video** using the button
* Bootstrap ensures the layout works on all devices
